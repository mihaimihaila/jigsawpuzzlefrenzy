How to Contact Support

Author: Mihai, last modified: 24/07/2025


If you're experiencing an issue with the game and have reached out via email, we kindly ask that you also send us a message directly from within the game. This helps us access your account details and game data so we can investigate and resolve the issue more efficiently.

Please follow these simple steps:

  1. Open the game on your device.
  2. Tap Home to go to the main screen.
  3. Tap the Account icon in the top-right corner.
  4. Select Feedback from the menu.
  5. Tap Send to submit your message.

This message will automatically include important information about your account and game progress, which allows us to assist you faster and more accurately.
